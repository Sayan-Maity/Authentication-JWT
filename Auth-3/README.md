# Authentication (JWT)

## Tech Stacks Used
- ReactJS (npx)
- Local Storage
- Google Signin
- NodeJS
- ExpressJS
- MongoDB

## Routes Used
Frontend :
Public routes
- ```/```                    : Landing page 
- ```/login```               : Login page 
- ```/register```            : Register page 
Private routes
- ```/addTour```             : Add Tour Page 
- ```/editTour/:id```        : Edit Tour Page 
- ```/dashboard```            : Dashboard 

Backend :
- ```/users/googleSignIn```               : 
- ```/users/signup```          : login route
- ```/users/signin```      : register route
