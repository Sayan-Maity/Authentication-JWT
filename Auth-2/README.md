# Authentication (JWT)

## Tech Stacks Used
- ReactJS
- Cookies
- NodeJS
- ExpressJS
- MongoDB

## Routes Used
Frontend Routes :
- ```/```                  : Demo Private Page
- ```/demoPrivate2```      : Demo Private 2 Page
- ```/demoPublic```        : Demo Public Page
- ```/demoPublic2```       : Demo Public 2 Page
- ```/login```             : Login Page
- ```/register```          : Register Page

Backend Routes :
Auth Routes
- ```/auth/register```   : Register a new user
- ```/auth/login```      : Login a user
- ```/```                : check User
